# SCT_CS_1
This is a Python program that encrypts and decrypts messages using the Caesar Cipher algorithm.

 Description:
 
Encrypts text using a user-defined shift value

Decrypts encrypted text back to readable form

Handles both uppercase and lowercase letters

Keeps numbers, spaces, and punctuation unchanged

Simple and beginner-friendly Python code

How It Works:

The user enters a message.

The program shifts each alphabetic character by the number specified by the user.

The user can choose to encrypt or decrypt the message.
